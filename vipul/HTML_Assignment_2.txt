Q 2.2 Why is HTML called a mark-up language?
-> A markup language is a type of language used to annotate text and  it marks the information by tagging them.
Example: <p> hello </p>
Here its marking the word “hello” by <p> paragraph tag. So that browser will know how to display it.

Q 2.3 Describe one problem you would likely face if you included presentation (styling) information with HTML instead of separating it in CSS?
-> If I want to style same in different web pages, I have to style that every text which I want same styling. Instead of this it is better to make one css file for all text which you want same styling.
